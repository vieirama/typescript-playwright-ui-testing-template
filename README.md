# typescript-playwright-ui-testing-template

```npm run run_test```


Template for testing framework using Typescript and Playwright.

## Installation

### Install dependencies

```commandline
npm install
```

### Install playwright

```commandline
npx playwright install
```

## Local execution

Headless:

```commandline
npm run run_test
```

Or headed execution:

```commandline
npm run run_test:headed
```
